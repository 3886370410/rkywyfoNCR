# 项目简介

## 核心功能
- 前后端代码及SQL文件
- 万字文档，包含答辩PPT
- 参考导入调试视频，讲解视频，开发工具安装包

## 适用场景
- 房产租赁行业
- 企业或个人开发房产租赁系统

## 技术特点
- B/S架构，MVC模式
- 使用Java、SpringBoot、Vue等技术栈
- 开发环境：IDEA、JDK1.8、Mysql、Maven、Navcat
- 支持管理员、房东、用户三个角色
- 支持私聊房东等特色功能
- 提供文档用例图，流程图，功能结构图，ER图
